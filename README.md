# Idea-manager

## Description

manager for your ideas

## Download

Python 3.8

```bash
$ git clone https://github.com/peach-lasagna/Idea-manager.git
$ cd Idea-manager
```
make venv in project:
```bash
$ poetry config virtualenvs.in-project true --local
```
Update dependies:
```bash
$ poetry lock
```
Install dependies:
```bash
$ poetry install
```

or

```bash
$ pip install requirements.txt
```
